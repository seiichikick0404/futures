## futures開発理由
<p align="center">
  こちらを作った背景は、何か新しいスキルや知識を付けたいと考えている人がtwitterなどで毎日1日の積み上げを発信してますが、かなりの人は挫折してしまっている事に気が付き何かできる事はないか？と考え、futuresを開発しました！
  futuresのミッションは継続し自分の未来をより良くするです。そのために3つの要素を盛り込むことを決めました！
  ゲーム✖️繋がり✖️手軽です。学習時間の記録、ランキング機能はゲームにあたり。繋がりはSNS要素である投稿を他ユーザーと共有することです。最後に手軽は投稿までの項数をできるだけ少なくし、投稿へのハードルを下げるようにしています。
</p>

## 実装予定の機能
  ・レベルアップ機能<br>

  ・学習時間のグラフ化<br>
